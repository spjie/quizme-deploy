# AI Flashcard Generator

An app that turns your notes into AI-generated flashcards to make studying more efficient.

## Features
- Paste your notes or select a subject
- Customize difficulty level, question type, and 
- Automatically generate flashcards using AI
- Simple interface for reviewing cards

## Tech Stack
- Next.js
- TypeScript
- Tailwind CSS

## Demo
[https://studywithquizme.vercel.app/](url)

## Setup
npm install
npm run dev
